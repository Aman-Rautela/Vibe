# Vibe

## Features
- Sign In / Sign Up with Firebase
- TailwindCSS UI
- Responsive Layout

## Setup
1. Clone the repo  
   ```bash
   git clone git@github.com:Aman-Rautela/Vibe.git
   cd Vibe
   ```
2. Install dependencies  
   ```bash
   npm install
   ```
3. Run the project  
   ```bash
   npm run dev
   ```
